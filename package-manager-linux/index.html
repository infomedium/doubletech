    Linux Package Managers

    Types

    *.deb file
    Debian-based systems
        Debian
        Kali
        Ubuntu      
    dpkg (Debian Package) (1994)
        > dpkg <options> actions
        - low-level utility
    apt-get (APT package handling utility) (1998)
        > apt-get [options]
    apt (Advanced Package Tool) (2014)
        > apt <options> actions
        - high-level utility
        - default package manager tool

*.rpm file  
    RedHat-based systems
        CentOS
        Fedora
        RHEL
    rpm (Red hat Package Manager) (1997)
        > rpm <options> actions
        - low-level utility
    yum (Yellowdog Updater Modified) (2002)
        > yum [options] [command] [package(s)]
        - high-level utility
    dnf (Dandified YUM) (2012)
        > dnf [options] <command> [<args>...]
        - successor to YUM
        - available starting from RHEL 8

    Useful commands - high Level

    S# Info apt yum dnf
    1 Fetches the list of available updates sudo apt update

    apt list --upgradable    		 yum check-update

    yum update ?    		 dnf list updates
    2 Installs some updates; does not remove packages sudo apt upgrade yum upgrade dnf upgrade
    3 Installs updates; may also remove some packages, if needed sudo apt full-upgrade
    4 Removes any old packages that are no longer needed sudo apt autoremove sudo yum autoremove sudo dnf autoremove
    5 Search in package description apt search <text to search e.g. > yum search <text to search e.g. > dnf search <text to search e.g. >
    6 Show details of a package apt show <package name e.g. npm> yum info <package name e.g. npm> dnf info <package name e.g. npm>
    Show changelog of a package apt changelog <package name e.g. npm> yum changelog <package name e.g. npm> dnf changelog <package name e.g. npm>
    7 List packages with matching name apt list <package name e.g. npm*> yum list <package name e.g. npm*> dnf list <package name e.g. npm*>
    List packages which are installed apt list --installed yum list --installed dnf list --installed
    8 Install a package sudo apt install <package name e.g. npm> sudo yum install <package name e.g. npm> sudo dnf install <package name e.g. npm>
    9 Reinstall a package sudo apt reinstall <package name e.g. npm> sudo yum reinstall <package name e.g. npm> sudo dnf reinstall <package name e.g. npm>
    10 Remove a package sudo apt remove <package name e.g. npm> sudo yum remove <package name e.g. npm> sudo dnf remove <package name e.g. npm>
    11 Show transaction history cat /var/log/apt/history.log yum history dnf history
    12 Rollback yum history rollback <ID> dnf history rollback <ID>

    Useful commands - low level

    S# Info dpkg rpm
    1 List files of package installed dpkg --listfiles <package name e.g. npm> rpm -ql <package name e.g. npm>

    Appendix

    Does apt autoremove remove all the unused packages?

    apt list --installed | wc -l
# 983

sudo apt install vlc

apt list --installed | wc -l
# 1162

sudo apt remove vlc

apt list --installed | wc -l
# 1161

sudo apt autoremove

apt list --installed | wc -l
# 984
